# 编写调用自身的函数变量

当我尝试这样编写递归函数变量时，编译器会报错：

```go
func main() {
    call := func(n int){
        if n > 0 {
            fmt.Println(n)
            call(n - 1) // 编译器会报错
        }
    }
}
```

实际上是可以的，只需要先定义这个函数变量：

```go
func main() {
    var call func (int)
    call = func(n int){
        if n > 0 {
            fmt.Println(n)
            call(n - 1)
        }
    }
}
```
