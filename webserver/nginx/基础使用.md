# 基础使用

## 安装

[安装Nginx](https://nginx.org/en/linux_packages.html)

## 默认目录

Nginx使用以下默认目录

```
# 配置目录
/etc/nginx
# 主配置文件
/etc/nginx/nginx.conf
# 网站配置文件，由主配置文件include
/etc/nginx/conf.d/*.conf

# 静态网页文件目录
/usr/share/nginx
```
## 命令行

### 生命周期

通过`nginx -s command`向nginx发送生命周期信号，常用的包括：

```
# 启动服务
nginx -s start
# 重启服务
nginx -s reload
# 关闭服务
nginx -s stop
# 测试配置文件
nginx -t
```
