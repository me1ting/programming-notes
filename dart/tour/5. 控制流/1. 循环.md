# 循环

## for

支持传统的`for`语法 和 `for range`语法：

```dart
for (var i = 0; i < 2; i++) {
  callbacks.add(() => print(i));
}

for (final c in callbacks) {
  c();
}
```

### 闭包优化

闭包捕获循环变量是一个经典问题，在JavaScript中出现较多，但在其它语言中也出现，细节可以参考笔记《由一个let问题而对javascript的复习》。

Dart采取了与JavaScript相同的解决方案，每次执行循环体，创建的的循环变量都是一个新的副本。

```dart
var callbacks = [];
for (var i = 0; i < 2; i++) {
  callbacks.add(() => print(i));
}

for (final c in callbacks) {
  c();
}
```
