# 概览

基本上来说是JavaScript异步那一套：`async`定义异步函数,`await`等待异步操作执行完成,`Future`表示异步这一行为。

唯一的区别只是语法形式的不同，`dart`将`async`关键字放在后面：

```dart
void main() async {
  checkVersion();
  print('In main: version is ${await lookUpVersion()}');
}
```
