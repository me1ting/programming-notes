# 基本概念

## Python的两种执行模式

和所有脚本语言一样，Python也有2种执行模式：

- 交互式
- 脚本

### 交互式

当我们在命令行执行`python`或`python3`(linux)，就进入了交互式执行模式。

交互式执行模式一般用来当作计算器，或者执行简单的代码并打印输出。

### 脚本

单文件的Python代码，可以直接被解释器执行。

```bash
python myscript.py
```

特别的，模块也可以以脚本的方式执行：

```bash
python3 -m http.server
```

普通脚本文件位于**文件目录**下，而模块文件位于**模块路径**下，这是后者要使用`-m`参数的原因。
