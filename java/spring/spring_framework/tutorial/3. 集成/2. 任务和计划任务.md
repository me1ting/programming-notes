# 任务和计划任务

**任务**(task)通常指异步（虽然设计上允许同步）的执行逻辑。经典的Java线程是1:1物理线程，因此需要需要池化管理执行任务的线程。

计划任务(scheduled task)是定时执行的任务。执行计划任务的抽象一般称为调度器，调度器的执行逻辑应当是轻量的，而把真正的任务委托给任务执行器执行。

Spring框架对于以上两个需求都提供了相应的封装。
